# LinkGame
Android平台实现的一个有关LOL主题的练练看小游戏

## 说明：

基本算法取材自《疯狂Android》一书

同时增加了背景音乐和触发音效

可直接下载apk查看效果

*********************************************
	Android版	LOL 连连看

一，版本为1.0版，存在较多bug，后期版本会不断完善。

二，由于开发平台所限，有可能出现屏幕适应问题。

三，反馈建议会为我们提供帮助。

四，Apk直接安装即可。
